Projects

Table Topics

Table Topics is a website that allows users to stay up-to-date on current activities in Congress. This includes upcoming bills, recent vote role calls, and profiles of all members in the Senate and House. Users have the option to register which allows them to start conversations on representatives profiles and specific bills. Additionally, users can record video respones from their computers.

This was a solo project assignment completed during a two-week timeframe while attending Prime Digital Academy.

This website was built using the MEAN stack: MongoDB, Express.js, AngularJS, Node.js. Additionally, Amazon Web Services S3 and Heroku were used for file storage and hosting the application.

Cushman & Wakefield

This website application was built to replace physical brochures and reports that were being hand-deliverd to business contacts in the Minneapolis and St. Paul market. The charts and data displayed on the brochures have been translated to an online interactive dashboard that allows customers to easily engage with real estate market data. Additionally, the administrative functionality of the website allows Cushman & Wakefield employees to automate their email process and manage customer inquiries submitted on the website. The website application and email functionality were integreated with third-party tools, Google Analytics, Google Tag Manager, and MailChimp to track the complete customer journey and support on-going communication.

This was a group project assignment was completed during a three-week timeframe while attending Prime Digital Academy.

This website was built using the MEAN stack: MongoDB, Express.js, AngularJS, Node.js. Additionally, Amazon Web Services S3 and Heroku were used for file storage and hosting the application.
